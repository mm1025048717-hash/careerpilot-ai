```
AI: 好的！我理解你的需求：

    💼 岗位: 前端开发
    💰 薪资: 15-25K
```

```
    ✅ 发展空间大

AI: 好的！薪资谈判话术建议：
    
    📝 话术模板：
    
    "非常感谢贵司的offer！我很期待加入团队。
    
    关于薪资，我想真诚地和您沟通一下：
    
    1. 我目前的薪资是23K*13薪（实际可以适当提高）

    • 用数据和事实说话
    • 适当提及其他机会（但不要过度）
    • 给对方台阶，提供替代方案
    • 保持专业，不要过分纠结
    
    祝你谈判顺利！🍀
```

## 自定义配置示例

### 设置AI服务

```
1. 打开"设置"页面
2. 选择AI提供商: OpenAI
3. 填写API Key: sk-xxx...
4. API Endpoint: https://api.openai.com/v1
5. 模型: gpt-4
6. 保存设置
```

### 配置BOSS账号

```
1. 打开"设置"页面
2. 填写BOSS直聘账号: your_phone@example.com
3. 填写密码: ********
4. 保存设置
```

## 常用命令

- `投递简历` - 开始投递流程
- `自动回复` - 开启自动回复
- `停止任务` - 停止当前任务
- `查看进度` - 查看求职数据
- `优化简历` - 获取简历优化建议
- `面试安排` - 查看面试日程
- `帮助` - 显示功能列表
