# 🚀 快速开始指南

## Windows用户（推荐）

### 方式一：一键启动（最简单）

1. 双击 `启动项目.bat`
2. 等待自动安装依赖和启动服务（首次运行需要几分钟）
3. 浏览器打开 http://localhost:5173
4. 开始使用！

### 方式二：分别启动

1. 首次运行：双击 `backend/setup_env.bat` 安装依赖
2. 双击 `backend/run.bat` 启动后端
3. 双击 `frontend/run.bat` 启动前端
4. 浏览器打开 http://localhost:5173

## Mac/Linux用户

### 1. 安装依赖（首次运行）

\`\`\`bash
# 添加执行权限
chmod +x backend/setup_env.sh backend/run.sh frontend/run.sh

# 安装后端依赖
cd backend
./setup_env.sh
cd ..
\`\`\`

### 2. 启动后端

\`\`\`bash
cd backend
./run.sh
\`\`\`

### 3. 启动前端（新终端）

\`\`\`bash
cd frontend
./run.sh
\`\`\`

## 首次使用配置

### 1. 获取AI API Key

选择以下任意一个AI服务商：

#### Option A: OpenAI（推荐）
- 访问：https://platform.openai.com/api-keys
- 创建新的API Key
- 模型推荐：`gpt-4` 或 `gpt-3.5-turbo`

#### Option B: Anthropic Claude
- 访问：https://console.anthropic.com/
- 创建新的API Key
- 模型推荐：`claude-3-5-sonnet-20241022`

#### Option C: DeepSeek（性价比高）
- 访问：https://platform.deepseek.com/
- 创建新的API Key
- 模型：`deepseek-chat`

### 2. 配置应用

1. 打开浏览器访问 http://localhost:5173
2. 点击左侧"设置"
3. 填写配置：
   - 选择AI提供商
   - 填写API Key
   - 填写API Endpoint（通常自动填充）
   - 填写模型名称
4. 填写BOSS直聘账号（可选）
5. 点击"保存设置"

### 3. 开始使用

切换到"AI对话"标签，尝试以下对话：

## 常见问题

### Q1: 后端启动失败？

**A:** 检查Python版本：
\`\`\`bash
python --version  # 需要 3.8+
\`\`\`

如果版本过低，访问 https://www.python.org/downloads/ 下载最新版本。

### Q2: 前端启动失败？

**A:** 检查Node.js版本：
\`\`\`bash
node --version  # 需要 18+
\`\`\`

如果版本过低，访问 https://nodejs.org/ 下载LTS版本。

### Q3: 无法连接后端？

**A:** 确保：
1. 后端服务已启动（看到"Running on http://0.0.0.0:5000"）
2. 端口5000没有被占用
3. 防火墙允许本地连接

### Q4: AI没有响应？

**A:** 检查：
1. API Key是否正确填写
2. API Endpoint是否正确
3. 网络是否能访问AI服务（可能需要代理）
4. 查看浏览器控制台（F12）的错误信息

### Q5: BOSS直聘登录失败？

**A:** 
1. 首次使用建议手动登录
2. BOSS可能有验证码，需要人工处理
3. 不要频繁尝试登录，避免账号异常
