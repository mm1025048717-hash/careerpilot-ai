【Windows用户 - 推荐】

  方式A：一键启动（最简单）
  ┌────────────────────────────────────────┐
  │  1. 双击：启动项目.bat                 │
  │  2. 等待自动安装和启动（首次需3-5分钟）│
  │  3. 浏览器打开：http://localhost:5173  │
  └────────────────────────────────────────┘

  方式B：分别启动
  ┌────────────────────────────────────────┐
  │  1. 双击：backend\run.bat（启动后端）  │
  │  2. 双击：frontend\run.bat（启动前端） │
  │  3. 浏览器打开：http://localhost:5173  │
  └────────────────────────────────────────┘


【Mac/Linux用户】

  1. 打开终端，进入项目目录
  2. 添加执行权限：
     chmod +x backend/run.sh frontend/run.sh

  3. 启动后端（终端1）：
     cd backend && ./run.sh

  4. 启动前端（终端2）：
     cd frontend && ./run.sh

  5. 浏览器打开：http://localhost:5173

  常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1: 后端启动失败？
A:  检查Python版本（需要3.8+）
    命令：python --version

Q2: 前端启动失败？
A:  检查Node.js版本（需要18+）
    命令：node --version

Q3: 无法连接后端？
A:  确保后端服务已启动，端口5000未被占用
    后端地址：http://localhost:5000/api/health

Q4: AI没有响应？
A:  检查API Key是否正确
    检查网络是否能访问AI服务
    查看浏览器控制台（F12）的错误信息

Q5: BOSS直聘登录失败？
A:  首次使用建议手动登录
    可能需要处理验证码（人工介入）
    不要频繁尝试登录
